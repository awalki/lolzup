# LolzUP - blazing edition for slow dedicated servers

> Блейзинг версия для слабых дедиков подразумевает использование Rust для ускорения работы автоапа на слабом железе.

> [!WARNING]
> Функционал может быть очень примитивным, из-за использования низкоуровневой библиотеки для работы с Telegram BOT API и отсутствием нормальной библиотеки для планировки задач

## Список команд

`/new thread_id - создает задачу для поднятия темы, кулдауны выставлять не нужно`

`/del thread_id - удаляет задачу из шедулера`

`/list - active tasks`

## Сборка

Убедитесь что у вас установлен Rust - https://rust-lang.org/learn/get-started/

```bash
cargo build --release
```

## Настройка

```
BOT_TOKEN=токен бота @BotFather
ADMIN_ID=ваш тг айди
LOLZ_TOKEn=ваш лолз токен
```
